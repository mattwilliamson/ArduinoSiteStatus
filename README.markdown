Arduino Site Status
===================

Check a url with cron and show it's status on a RGB LED.


Crontab
-------

    *    *    *    *    *    ~/Dropbox/Projects/OpenSource/ArduinoSiteStatus/checkstatus.py http://google.com


Requirements
------------

* pyserial
* requests
